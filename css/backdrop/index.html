The backdrop-filter property accepts the same filter functions as the filter property. These include: blur(): Applies a Gaussian blur. brightness(): Adjusts the brightness. contrast(): Adjusts the contrast. grayscale(): Converts the image to grayscale. hue-rotate(): Rotates the hue of the colors. invert(): Inverts the colors. opacity(): Adjusts the opacity. saturate(): Adjusts the saturation. sepia(): Applies a sepia tone. drop-shadow(): Applies a drop shadow. filter transision animation transform (2d , 3d ) flex grid pseudo element pseudo classess
